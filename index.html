Search For Users

- As a regular user you are able to search for other users using one of these categories:

  • first name
  • last name
  • job title
  • department
  • locations

-There are currently only 5 locations to choose from:

  • London
  • New York
  • Paris
  • Munich
  • Rome

- To perform search, choose from a catefory from the select on the left. If the field is left empty, it will return all the personnel that matches that category.

Add new user

You can only add new users or departments as an admin.

Press the green + button next to the search button and fill in the required fields. You will be prompted with a success message on completion or an error on failure.
